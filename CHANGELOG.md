# Changelog

## 1.5.0 - 2026-07-07

- Added keyboard navigation in the picker grid: arrow keys move between icons, Home/End jump to the first/last result, `Enter` or `Space` selects

> This release also consolidates the changelog. Versions 1.0.0–1.4.1 covered the initial build-out:
> - Tabler Icon field type with a searchable picker — 6,000+ bundled outline and filled icons, category dropdown, and outline/filled tabs
> - Compact field input: clickable preview box, Choose/Remove buttons
> - Inline SVG templating with `size`, `strokeWidth`, and arbitrary attributes
> - Site-wide attribute defaults via `svgDefaults` in `config/tabler.php`
> - `tabler()` Twig function for manual icons, with `-filled` suffix support and Icon pass-through
